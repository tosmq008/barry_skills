# 测试用例模板

## 测试用例编写规范

### 用例 ID 命名规则

| 测试类型 | 前缀 | 示例 |
|----------|------|------|
| 单元测试 | TC_UNIT_ | TC_UNIT_001 |
| 集成测试 | TC_API_ | TC_API_001 |
| 黑盒测试 | TC_FUNC_ | TC_FUNC_001 |
| UI 自动化 | TC_UI_ | TC_UI_001 |
| 冒烟测试 | TC_SMOKE_ | TC_SMOKE_001 |

### 优先级定义

| 优先级 | 说明 | 执行要求 |
|--------|------|----------|
| P0 | 核心功能，阻塞性 | 必须执行，必须通过 |
| P1 | 重要功能，影响主流程 | 必须执行 |
| P2 | 一般功能，不影响主流程 | 建议执行 |
| P3 | 边缘功能，优化类 | 时间允许时执行 |
