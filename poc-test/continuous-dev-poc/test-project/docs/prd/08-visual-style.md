# 08 视觉风格规范

## 整体风格

- 简洁现代
- 居中布局
- 清晰易读

## 颜色规范

| 用途 | 颜色 | 色值 |
|------|------|------|
| 背景色 | 浅灰 | #f5f5f5 |
| 卡片背景 | 白色 | #ffffff |
| 主文字 | 深灰 | #333333 |
| 增加按钮 | 绿色 | #4CAF50 |
| 减少按钮 | 红色 | #f44336 |
| 按钮文字 | 白色 | #ffffff |

## 字体规范

| 元素 | 字体 | 大小 | 粗细 |
|------|------|------|------|
| 标题 | system-ui | 24px | bold |
| 数字显示 | system-ui | 72px | bold |
| 按钮文字 | system-ui | 32px | bold |

## 间距规范

| 元素 | 间距 |
|------|------|
| 卡片内边距 | 40px |
| 按钮间距 | 20px |
| 数字与按钮间距 | 30px |

## 按钮样式

| 属性 | 值 |
|------|------|
| 宽度 | 80px |
| 高度 | 80px |
| 圆角 | 50% (圆形) |
| 阴影 | 0 4px 6px rgba(0,0,0,0.1) |
| hover效果 | 轻微放大 + 加深颜色 |

## 响应式

- 移动端友好
- 按钮足够大，便于触摸
- 居中布局适应各种屏幕
