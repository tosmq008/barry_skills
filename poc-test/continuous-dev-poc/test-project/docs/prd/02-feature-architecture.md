# 02 功能架构图

## 系统架构

```
┌─────────────────────────────────────┐
│           计数器 App                 │
├─────────────────────────────────────┤
│                                     │
│  ┌─────────────────────────────┐   │
│  │        显示区域              │   │
│  │    ┌─────────────────┐      │   │
│  │    │   当前数字: 0    │      │   │
│  │    └─────────────────┘      │   │
│  └─────────────────────────────┘   │
│                                     │
│  ┌─────────────────────────────┐   │
│  │        操作区域              │   │
│  │  ┌─────┐       ┌─────┐      │   │
│  │  │  -  │       │  +  │      │   │
│  │  └─────┘       └─────┘      │   │
│  └─────────────────────────────┘   │
│                                     │
└─────────────────────────────────────┘
```

## 功能模块

| 模块 | 功能 | 说明 |
|------|------|------|
| 显示模块 | 展示当前计数值 | 居中显示大字体数字 |
| 增加模块 | 点击增加计数 | 点击 + 按钮，数字 +1 |
| 减少模块 | 点击减少计数 | 点击 - 按钮，数字 -1 |

## 数据流

```
用户点击 + → 触发 increment() → count += 1 → 更新显示
用户点击 - → 触发 decrement() → count -= 1 → 更新显示
```
