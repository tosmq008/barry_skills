# 05 交互页面清单

## 页面列表

由于这是一个单页面应用，只有一个页面：

| 页面编号 | 页面名称 | 路径 | 说明 |
|----------|----------|------|------|
| P001 | 计数器主页 | index.html | 唯一页面，包含所有功能 |

## 页面 P001 详情

### 页面元素

| 元素 | 类型 | 说明 |
|------|------|------|
| 标题 | 文本 | "计数器" |
| 数字显示 | 文本 | 显示当前计数值 |
| 减少按钮 | 按钮 | 点击减少计数 |
| 增加按钮 | 按钮 | 点击增加计数 |

### 页面状态

| 状态 | 说明 |
|------|------|
| 初始状态 | 数字显示 0 |
| 正数状态 | 数字 > 0 |
| 负数状态 | 数字 < 0 |
