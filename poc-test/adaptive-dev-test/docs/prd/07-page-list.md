## 2. 页面详细说明

### P01 - 登录页

**页面目标**: 用户登录系统

**页面元素:**
- Logo
- 标题："登录到待办事项"
- 邮箱输入框
- 密码输入框
- "记住我"复选框
- 登录按钮
- "忘记密码"链接
- "还没有账号？注册"链接

**交互行为:**
- 输入邮箱和密码
- 点击登录按钮
- 验证成功 → 跳转首页
- 验证失败 → 显示错误提示

**状态:**
- 默认状态
- 输入中状态（输入框聚焦）
- 加载状态（登录中）
- 错误状态（登录失败）

**异常处理:**
- 邮箱格式错误 → 提示"请输入有效的邮箱地址"
- 密码错误 → 提示"邮箱或密码错误"
- 网络错误 → 提示"网络连接失败，请重试"

---

### P02 - 注册页

**页面目标**: 新用户注册

**页面元素:**
- Logo
- 标题："创建账号"
- 用户名输入框
- 邮箱输入框
- 密码输入框
- 确认密码输入框
- 注册按钮
- "已有账号？登录"链接

**交互行为:**
- 输入用户名、邮箱、密码
- 实时校验密码强度
- 点击注册按钮
- 注册成功 → 自动登录并跳转首页
- 注册失败 → 显示错误提示

**状态:**
- 默认状态
- 输入中状态
- 加载状态（注册中）
- 成功状态
- 错误状态

**异常处理:**
- 邮箱已存在 → 提示"该邮箱已被注册"
- 密码不一致 → 提示"两次密码输入不一致"
- 密码强度不足 → 提示"密码至少8位，包含字母和数字"

- 分隔线

## 3. 页面状态汇总

### 3.1 通用状态

| 状态 | 说明 | 设计要点 |
|------|------|----------|
| **默认状态** | 正常显示 | 清晰的信息层级 |
| **加载状态** | 数据加载中 | 骨架屏或加载动画 |
| **空状态** | 无数据 | 插画 + 引导文案 |
| **错误状态** | 加载失败 | 错误提示 + 重试按钮 |

### 3.2 任务状态

| 状态 | 视觉表现 | 交互 |
|------|----------|------|
| **待办** | 空心复选框 | 点击标记完成 |
| **进行中** | 半选复选框 | 点击标记完成 |
| **已完成** | 实心复选框 + 删除线 | 点击取消完成 |
| **过期** | 红色标记 | 提示过期 |

## 4. 响应式设计

### 4.1 桌面端（>1024px）

- 三栏布局
- 侧边栏固定宽度 240px
- 详情面板固定宽度 360px
- 主内容区自适应

### 4.2 平板端（768px - 1024px）

- 侧边栏可折叠
- 详情面板覆盖在主内容区上方
- 主内容区全宽

### 4.3 移动端（<768px）

- 单栏布局
- 侧边栏抽屉式
- 详情面板全屏弹窗
- 底部导航栏
