# Alpha因子研究规范

## 1. 因子研究方法论

### 1.1 研究流程 (BlackRock/Two Sigma标准)

```
经济假设 ──▶ 数据准备 ──▶ 因子构建 ──▶ 单因子测试 ──▶ 因子合成 ──▶ 组合构建
```

### 1.2 因子构建原则

| 原则 | 说明 |
|------|------|
| 经济直觉 | 因子必须有经济学解释 |
| 数据可得 | 使用公开可获取数据 |
| 可复制 | 计算方法清晰可复制 |
| 时效性 | 考虑数据发布延迟 |

## 2. 因子检验方法

### 2.1 IC分析

```python
# IC计算示例
IC = correlation(factor_t, return_t+1)
IC_mean = mean(IC_series)
IC_std = std(IC_series)
IR = IC_mean / IC_std
```

### 2.2 分组回测

| 分组 | 方法 | 分析重点 |
|------|------|----------|
| 5分组 | 按因子值分5组 | 单调性 |
| 10分组 | 按因子值分10组 | 精细区分 |
| 多空组合 | Top组-Bottom组 | 因子收益 |

### 2.3 因子衰减分析

| 滞后期 | IC | 说明 |
|--------|-----|------|
| 1期 | 0.05 | 即时预测 |
| 5期 | 0.03 | 短期有效 |
| 20期 | 0.01 | 长期衰减 |

## 3. 因子合成方法

### 3.1 等权合成

```
合成因子 = (因子1 + 因子2 + ... + 因子N) / N
```

### 3.2 IC加权

```
权重_i = IC_i / sum(IC)
合成因子 = sum(权重_i × 因子_i)
```

### 3.3 最优化加权

```
max: 组合IC
s.t.: sum(权重) = 1
      权重 >= 0
```

## 4. 因子正交化

### 4.1 为什么要正交化

- 消除因子间相关性
- 避免重复暴露
- 提高组合效率

### 4.2 正交化方法

| 方法 | 公式 | 适用场景 |
|------|------|----------|
| 回归残差 | 因子 - β×基准因子 | 去除已知因子 |
| PCA | 主成分分解 | 降维 |
| Gram-Schmidt | 逐步正交 | 多因子 |
