## 2. 回测模式

| 模式 | 描述 | 适用场景 |
|------|------|----------|
| 向量化回测 | 批量计算 | 快速验证 |
| 事件驱动 | 逐笔模拟 | 精确回测 |
| 实时模拟 | 模拟实盘 | 上线前验证 |

## 3. 撮合规则

### 3.1 成交价格

| 模式 | 成交价 | 真实度 |
|------|--------|--------|
| 收盘价 | 当日收盘 | 低 |
| 开盘价 | 次日开盘 | 中 |
| VWAP | 成交量加权 | 高 |
| 滑点模拟 | 价格+滑点 | 最高 |

### 3.2 成交量限制

| 限制 | 规则 | 说明 |
|------|------|------|
| 涨跌停 | 不可成交 | 模拟真实 |
| 成交量 | <日成交20% | 避免冲击 |
| 最小单位 | 100股整数 | A股规则 |

## 4. 绩效指标计算

### 4.1 收益指标

```python
# 年化收益率
annual_return = (1 + total_return) ** (252 / days) - 1

# 超额收益
alpha = strategy_return - benchmark_return
```

### 4.2 风险指标

```python
# 最大回撤
max_drawdown = (peak - trough) / peak

# 夏普比率
sharpe = (return - rf) / volatility * sqrt(252)
```
