# 绩效指标规范

## 1. 收益类指标

| 指标 | 公式 | 说明 | 优秀标准 |
|------|------|------|----------|
| 累计收益率 | (Pn-P0)/P0 | 总收益 | >50%/年 |
| 年化收益率 | (1+R)^(252/n)-1 | 年化后收益 | >15% |
| 超额收益 | Rs-Rb | 相对基准 | >5% |
| 信息比率 | α/TE | 主动管理能力 | >0.5 |

## 2. 风险类指标

| 指标 | 公式 | 说明 | 优秀标准 |
|------|------|------|----------|
| 波动率 | σ×√252 | 年化波动 | <20% |
| 最大回撤 | max(Pi-Pj)/Pi | 最大亏损 | <15% |
| 下行波动率 | σ(负收益) | 下行风险 | <12% |
| VaR(95%) | 分位数损失 | 极端风险 | <3% |

## 3. 风险调整收益

| 指标 | 公式 | 说明 | 优秀标准 |
|------|------|------|----------|
| 夏普比率 | (R-Rf)/σ | 单位风险收益 | >1.5 |
| 索提诺比率 | (R-Rf)/σd | 下行风险调整 | >2.0 |
| 卡玛比率 | R/MDD | 回撤调整收益 | >1.0 |
| 特雷诺比率 | (R-Rf)/β | 系统风险调整 | >0.1 |

## 4. 交易类指标

| 指标 | 公式 | 说明 | 优秀标准 |
|------|------|------|----------|
| 胜率 | 盈利次数/总次数 | 交易成功率 | >50% |
| 盈亏比 | 平均盈利/平均亏损 | 风险回报 | >1.5 |
| 最大连亏 | 连续亏损次数 | 心理压力 | <5次 |
| 换手率 | 交易额/资产 | 交易频率 | 适中 |

## 5. 归因分析

### 5.1 收益归因

| 归因项 | 说明 |
|--------|------|
| 择时收益 | 仓位调整贡献 |
| 选股收益 | 个股选择贡献 |
| 行业配置 | 行业偏离贡献 |
| 风格暴露 | 因子暴露贡献 |

### 5.2 风险归因

| 归因项 | 说明 |
|--------|------|
| 市场风险 | Beta暴露 |
| 特异风险 | 个股特有 |
| 因子风险 | 风格因子 |
| 行业风险 | 行业集中 |

## 6. 指标计算周期

| 周期 | 用途 | 注意事项 |
|------|------|----------|
| 日度 | 实时监控 | 噪音较大 |
| 周度 | 短期评估 | 平滑处理 |
| 月度 | 定期报告 | 标准周期 |
| 年度 | 长期评估 | 完整周期 |
