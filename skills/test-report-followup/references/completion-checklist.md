# 完成检查清单

## 跟进完成标准

### 必须满足条件 (Must)

```markdown
## 必须条件检查

### Bug 修复验证
- [ ] 所有 Critical Bug 已修复并验证通过
- [ ] 所有 Major Bug 已修复并验证通过
- [ ] 无新增 Critical/Major Bug

### 测试通过率
- [ ] 整体测试通过率 ≥ 95%
- [ ] P0 用例 100% 通过
- [ ] P1 用例 ≥ 98% 通过

### 遗留问题
- [ ] 无阻塞性遗留问题
- [ ] 所有遗留问题有明确的规避方案
- [ ] 遗留问题已记录并获得确认
```

### 应该满足条件 (Should)

```markdown
## 应该条件检查

### Bug 修复
- [ ] 所有 Minor Bug 已修复或有明确遗留原因
- [ ] Trivial Bug 修复率 ≥ 80%

### PRD 问题
- [ ] 所有 PRD 问题已澄清
- [ ] PRD 文档已更新（如需要）
- [ ] 开发团队理解一致

### 体验问题
- [ ] 主要体验问题已改进
- [ ] 有明确的后续改进计划
```

### 可选满足条件 (Could)

```markdown
## 可选条件检查

### 优化类
- [ ] 所有 Trivial Bug 已修复
- [ ] 体验优化建议已实施
- [ ] 性能优化已完成

### 功能完善
- [ ] 缺失功能已评估并排期
- [ ] 行业对标差距已有改进计划
```

## 质量门禁

### 可发布条件

```markdown
## 发布质量门禁

### 必须通过
- [ ] Critical Bug: 0 个
- [ ] Major Bug: 0 个
- [ ] 测试通过率 ≥ 95%
- [ ] P0 用例 100% 通过

### 有条件通过
- [ ] Minor Bug ≤ 5 个（有规避方案）
- [ ] 测试通过率 ≥ 90%
- [ ] 遗留问题有明确计划

### 不可发布
- [ ] 存在 Critical/Major Bug
- [ ] 测试通过率 < 90%
- [ ] 存在阻塞性遗留问题
```
