# 任务单模板

## 通用任务单结构

```markdown
# [角色] 任务单

## 任务信息
| 项目 | 值 |
|------|-----|
| 来源报告 | test-report-v[x.x.x].md |
| 分派日期 | [日期] |
| 分派人 | 测试报告跟进专家 |
| 任务数量 | [n] |
| 截止日期 | [日期] |

---

## 任务列表

### 任务 [序号]: [问题ID] - [问题标题]

#### 问题信息
| 字段 | 值 |
|------|-----|
| 问题ID | [ID] |
| 问题类型 | Bug / PRD问题 / 体验问题 / 缺失功能 |
| 严重程度 | Critical / Major / Minor / Trivial |
| 优先级 | P0 / P1 / P2 / P3 |
| 来源 | 测试报告 - [章节名] |

#### 问题描述
[详细的问题描述，包括复现步骤、截图等]

#### 期望输出
- [ ] [输出项1]
- [ ] [输出项2]
- [ ] [输出项3]

#### 验收标准
- [验收标准1]
- [验收标准2]

#### 截止时间
- 开始时间: [日期]
- 截止时间: [日期]

#### 备注
[其他需要说明的信息]

---
```
