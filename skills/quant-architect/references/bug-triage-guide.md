# Bug分类处理指南

## 1. Bug分类标准（量化视角）

### 1.1 严重程度分类

| 级别 | 定义 | 示例 | 响应时间 |
|------|------|------|----------|
| Critical | 导致资金损失或系统不可用 | 错误下单、风控失效 | 立即 |
| Major | 影响核心功能但可绕过 | 策略信号错误、延迟异常 | 4小时 |
| Minor | 影响非核心功能 | 报表错误、日志缺失 | 24小时 |
| Trivial | 体验问题 | UI显示、文档错误 | 下版本 |

### 1.2 Bug类型分类

| 类型 | 说明 | 典型问题 |
|------|------|----------|
| 交易逻辑 | 订单处理相关 | 重复下单、订单丢失 |
| 数据处理 | 行情/因子相关 | 数据丢失、计算错误 |
| 风控相关 | 风控规则相关 | 规则未触发、误触发 |
| 性能问题 | 延迟/吞吐相关 | 延迟抖动、吞吐下降 |
| 配置问题 | 参数配置相关 | 参数错误、配置丢失 |

## 2. 根因分析方法

### 2.1 分析流程

```
┌─────────────────────────────────────────┐
│            Bug根因分析流程               │
├─────────────────────────────────────────┤
│  1. 问题复现                             │
│     └─ 确定触发条件和复现步骤            │
│  2. 日志分析                             │
│     └─ 追踪执行路径和异常点              │
│  3. 数据验证                             │
│     └─ 检查输入输出数据正确性            │
│  4. 代码审查                             │
│     └─ 定位问题代码                      │
│  5. 影响评估                             │
│     └─ 评估业务影响范围                  │
└─────────────────────────────────────────┘
```

### 2.2 常用分析工具

| 工具 | 用途 | 使用场景 |
|------|------|----------|
| 日志系统 | 追踪执行流程 | 逻辑问题 |
| 监控系统 | 查看指标变化 | 性能问题 |
| 调试器 | 单步调试 | 复杂逻辑 |
| Profiler | 性能分析 | 性能瓶颈 |

## 3. 量化系统特有问题

### 3.1 交易逻辑问题

**常见问题：**

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 重复下单 | 状态管理错误 | 幂等性设计 |
| 订单丢失 | 网络/队列问题 | 重试+对账 |
| 成交错误 | 回报处理错误 | 状态机校验 |

**排查步骤：**
1. 检查订单状态流转日志
2. 对比本地和交易所订单
3. 检查网络连接状态
4. 验证订单处理逻辑

### 3.2 数据问题

**常见问题：**

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 行情丢失 | 网络/解析错误 | 数据校验+补数据 |
| 数据延迟 | 处理瓶颈 | 性能优化 |
| 计算错误 | 算法/精度问题 | 单元测试 |

### 3.3 性能问题

**常见问题：**

| 问题 | 症状 | 排查方向 |
|------|------|----------|
| 延迟抖动 | P99延迟高 | GC/锁竞争 |
| 吞吐下降 | TPS降低 | 资源瓶颈 |
| 内存泄漏 | 内存持续增长 | 对象未释放 |

```markdown
3. [步骤3]

[预期的正确行为]

[防止类似问题的措施]
```

## 5. 修复优先级评估

### 5.1 优先级矩阵

| 影响\紧急 | 高 | 中 | 低 |
|-----------|----|----|-----|
| 高 | P0 | P1 | P2 |
| 中 | P1 | P2 | P3 |
| 低 | P2 | P3 | P4 |

### 5.2 量化系统特殊考量

**优先处理：**
- 涉及资金安全的问题
- 影响交易执行的问题
- 风控失效的问题

**可延后处理：**
- 非交易时段的问题
- 有临时绕过方案的问题
- 仅影响报表/日志的问题
