# 战略分析框架

## 1. SWOT 分析框架

### 1.1 SWOT 矩阵

```
              内部因素                    外部因素
         ┌─────────────────┐        ┌─────────────────┐
  有利   │   Strengths     │        │  Opportunities  │
  因素   │     优势        │        │      机会       │
         └─────────────────┘        └─────────────────┘
         ┌─────────────────┐        ┌─────────────────┐
  不利   │   Weaknesses    │        │    Threats      │
  因素   │     劣势        │        │      威胁       │
         └─────────────────┘        └─────────────────┘
```

```markdown

| [优势2] | [说明] | 高/中/低 |

### Weaknesses (劣势)

|--------|------|----------|
| [劣势1] | [说明] | 高/中/低 |
| [劣势2] | [说明] | 高/中/低 |

### Opportunities (机会)
| 机会项 | 说明 | 把握难度 |
|--------|------|----------|
| [机会1] | [说明] | 高/中/低 |
| [机会2] | [说明] | 高/中/低 |

### Threats (威胁)
| 威胁项 | 说明 | 威胁程度 |

```

### 1.3 SWOT 战略矩阵

| 策略 | 说明 | 适用情况 |
|------|------|----------|
| SO策略 | 利用优势抓住机会 | 优势明显+机会多 |
| WO策略 | 克服劣势抓住机会 | 机会多但有劣势 |
| ST策略 | 利用优势应对威胁 | 有优势但威胁大 |
| WT策略 | 减少劣势规避威胁 | 劣势多+威胁大 |

## 5. 风险评估框架

### 5.1 风险矩阵

```
影响程度
    ▲
高  │  中风险  │  高风险  │  极高风险
    │─────────┼──────────┼──────────
中  │  低风险  │  中风险  │  高风险
    │─────────┼──────────┼──────────
低  │  极低风险│  低风险  │  中风险
    └─────────┴──────────┴──────────▶ 发生概率
         低        中         高
```

### 5.2 风险评估模板

```markdown
## 风险评估: [项目名称]

### 风险清单
| 风险ID | 风险描述 | 类型 | 概率 | 影响 | 风险等级 | 缓解措施 |
|--------|----------|------|------|------|----------|----------|
| R001 | [描述] | 市场/技术/运营/财务/法规 | 高/中/低 | 高/中/低 | [等级] | [措施] |

### 风险应对策略
| 策略 | 说明 | 适用情况 |
|------|------|----------|
| 规避 | 消除风险来源 | 风险不可接受 |
| 转移 | 转移给第三方 | 可通过保险/外包 |
| 减轻 | 降低概率或影响 | 可采取措施 |
| 接受 | 接受风险 | 风险可接受 |
```
