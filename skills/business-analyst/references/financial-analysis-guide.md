# 财务分析指南

## 1. 关键财务指标

### 1.1 单位经济指标

| 指标 | 全称 | 计算公式 | 健康标准 |
|------|------|----------|----------|
| CAC | Customer Acquisition Cost | 营销费用 / 新增客户数 | 行业平均以下 |
| LTV | Lifetime Value | ARPU × 客户生命周期 | > 3x CAC |
| LTV/CAC | 单位经济效益 | LTV ÷ CAC | > 3 |
| ARPU | Average Revenue Per User | 总收入 / 用户数 | 持续增长 |
| Payback Period | 回本周期 | CAC / (ARPU × 毛利率) | < 12个月 |

### 1.2 盈利能力指标

| 指标 | 计算公式 | 说明 |
|------|----------|------|
| 毛利率 | (收入 - 成本) / 收入 | SaaS > 70%, 电商 > 20% |
| 净利率 | 净利润 / 收入 | 行业差异大 |
| EBITDA利润率 | EBITDA / 收入 | 排除非经营因素 |
| ROI | 净利润 / 投资额 | > 15% 为佳 |
| ROE | 净利润 / 股东权益 | > 15% 为佳 |

### 1.3 增长指标

| 指标 | 计算公式 | 说明 |
|------|----------|------|
| MRR | 月度经常性收入 | SaaS 核心指标 |
| ARR | 年度经常性收入 | MRR × 12 |
| MoM Growth | (本月-上月) / 上月 | 月环比增长 |
| YoY Growth | (今年-去年) / 去年 | 年同比增长 |
| CAGR | (终值/初值)^(1/n) - 1 | 复合年增长率 |
