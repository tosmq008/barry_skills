## 2. 九大模块详解

### 2.1 客户细分 (Customer Segments - CS)

**核心问题：** 我们为谁创造价值？谁是我们最重要的客户？

**客户类型：**
| 类型 | 说明 | 示例 |
|------|------|------|
| 大众市场 | 不区分客户群体 | 消费电子 |
| 利基市场 | 特定细分市场 | 专业软件 |
| 细分化 | 略有不同的需求 | 银行零售/私行 |
| 多元化 | 完全不同的客户群 | 亚马逊 |
| 多边平台 | 相互依存的客户群 | 信用卡 |

**填写模板：**
```markdown
### 客户细分

#### 主要客户群
- 客户群1：[描述]
  - 特征：[特征]
  - 规模：[规模]
  - 需求：[需求]

#### 次要客户群
- 客户群2：[描述]
```

### 2.2 价值主张 (Value Propositions - VP)

**核心问题：** 我们为客户提供什么价值？解决什么问题？

**价值类型：**
| 类型 | 说明 |
|------|------|
| 新颖性 | 满足全新需求 |
| 性能 | 提升产品/服务性能 |
| 定制化 | 满足个性化需求 |
| 完成任务 | 帮助客户完成工作 |
| 设计 | 卓越的设计 |
| 品牌/地位 | 品牌价值 |
| 价格 | 更低的价格 |
| 成本削减 | 帮助客户降低成本 |
| 风险降低 | 降低客户风险 |
| 可达性 | 让更多人获得 |
| 便利性 | 更方便使用 |

### 2.3 渠道通路 (Channels - CH)

**核心问题：** 如何触达客户？如何传递价值主张？

**渠道阶段：**
| 阶段 | 说明 | 渠道类型 |
|------|------|----------|
| 认知 | 如何让客户知道我们 | 广告、PR、SEO |
| 评估 | 如何帮助客户评估 | 官网、试用、评测 |
| 购买 | 客户如何购买 | 电商、直销、代理 |
| 传递 | 如何传递价值 | 物流、下载、服务 |
| 售后 | 如何提供售后支持 | 客服、社区、培训 |

### 2.4 客户关系 (Customer Relationships - CR)

**核心问题：** 与客户建立什么类型的关系？

**关系类型：**
| 类型 | 说明 | 成本 |
|------|------|------|
| 个人助理 | 人工服务 | 高 |
| 专属服务 | 专属客户经理 | 很高 |
| 自助服务 | 客户自助 | 低 |
| 自动化服务 | 智能服务 | 中 |
| 社区 | 用户社区 | 中 |
| 共同创造 | 用户参与创造 | 中 |

### 2.5 收入来源 (Revenue Streams - R$)

**核心问题：** 客户愿意为什么付费？如何付费？

**收入类型：**
| 类型 | 说明 | 示例 |
|------|------|------|
| 资产销售 | 出售产品所有权 | 零售 |
| 使用费 | 按使用量收费 | 云服务 |
| 订阅费 | 按周期收费 | SaaS |
| 租赁费 | 临时使用权 | 租车 |
| 授权费 | 知识产权授权 | 专利 |
| 经纪费 | 中介服务费 | 房产中介 |
| 广告费 | 广告收入 | 媒体 |

### 2.6 关键资源 (Key Resources - KR)

**核心问题：** 需要什么关键资源来支撑商业模式？

**资源类型：**
| 类型 | 说明 | 示例 |
|------|------|------|
| 实体资源 | 物理资产 | 设备、厂房、网络 |
| 知识资源 | 知识产权 | 专利、品牌、数据 |
| 人力资源 | 人才团队 | 研发、销售、运营 |
| 财务资源 | 资金 | 现金、信贷、股权 |

### 2.7 关键活动 (Key Activities - KA)

**核心问题：** 需要做什么关键活动来运营商业模式？

**活动类型：**
| 类型 | 说明 | 示例 |
|------|------|------|
| 生产 | 产品设计、制造、交付 | 制造业 |
| 问题解决 | 为客户解决问题 | 咨询、医疗 |
| 平台/网络 | 平台运营、网络维护 | 互联网平台 |

### 2.8 关键合作 (Key Partnerships - KP)

**核心问题：** 谁是关键合作伙伴？需要什么关键资源？

**合作类型：**
| 类型 | 说明 | 目的 |
|------|------|------|
| 战略联盟 | 非竞争者之间 | 资源互补 |
| 竞合关系 | 竞争者之间 | 共同利益 |
| 合资企业 | 共同投资 | 新业务开发 |
| 供应商关系 | 供应链合作 | 资源获取 |

### 2.9 成本结构 (Cost Structure - C$)

**核心问题：** 商业模式的主要成本是什么？

**成本类型：**
| 类型 | 说明 | 特点 |
|------|------|------|
| 固定成本 | 不随产量变化 | 租金、工资 |
| 可变成本 | 随产量变化 | 原材料、佣金 |
| 规模经济 | 规模越大成本越低 | 采购、生产 |
| 范围经济 | 范围越广成本越低 | 多产品线 |
