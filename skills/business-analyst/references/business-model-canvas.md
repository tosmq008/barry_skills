# 商业模式画布（BMC）与VRIO联动模板

本模板用于系统性评估企业内部环境，为 SWOT 中的 Strengths / Weaknesses 提供结构化输入。

---

## 1. BMC 九要素总览

| 模块 | 核心问题 | 输出要点 |
|---|---|---|
| 客户细分（CS） | 我们服务谁？ | 核心客户群、需求差异、价值规模 |
| 价值主张（VP） | 我们解决什么问题？ | 核心痛点、差异化、可感知价值 |
| 渠道通路（CH） | 如何触达和交付？ | 获客、转化、交付、售后链路 |
| 客户关系（CR） | 如何维护关系？ | 拉新、留存、复购、NPS 机制 |
| 收入来源（R$） | 如何变现？ | 收入模型、价格逻辑、结构占比 |
| 关键资源（KR） | 必须拥有什么？ | 技术、数据、品牌、人才、资本 |
| 关键活动（KA） | 必须做好什么？ | 研发、运营、销售、服务、合规 |
| 重要伙伴（KP） | 谁帮助我们成功？ | 供应链、渠道、生态、战略伙伴 |
| 成本结构（C$） | 成本由什么构成？ | 固定/变动成本、杠杆点、盈亏平衡 |

---

## 2. 九要素详细填写指南

### 2.1 客户细分（CS）

**填写问题：**
1. 谁是付费决策者？谁是实际使用者？
2. 是否存在高价值细分（高ARPU/高复购/高传播）？
3. 各细分客户对“价格、效率、体验、风险”的偏好差异是什么？

### 2.2 价值主张（VP）

**填写问题：**
1. 你的产品替代了客户的哪种“旧方案”？
2. 价值是“更便宜”“更快”“更好”还是“更安全”？
3. 是否能被客户清晰感知并愿意付费？

### 2.3 渠道通路（CH）

**渠道五阶段：**认知 → 评估 → 购买 → 交付 → 售后。

**填写问题：**
1. 每个阶段的主要渠道是什么？
2. 渠道成本与转化率是否匹配？
3. 是否过度依赖单一渠道？

### 2.4 客户关系（CR）

**填写问题：**
1. 关系类型是交易型、订阅型还是社区型？
2. 关键留存机制是什么？
3. 客户成功体系是否可规模化？

### 2.5 收入来源（R$）

**常见模型：**一次性销售、订阅、交易抽佣、广告、授权、增值服务。

**填写问题：**
1. 收入是否过于单一？
2. 价格机制是否与客户价值一致？
3. 收入是否具备复利属性（续费/增购/网络效应）？

### 2.6 关键资源（KR）

**资源类别：**
- 实体资源：设备、产能、网络
- 知识资源：专利、算法、数据、品牌
- 人力资源：关键岗位、核心团队
- 财务资源：现金流、融资能力

### 2.7 关键活动（KA）

**填写问题：**
1. 哪些活动决定客户价值能否兑现？
2. 哪些活动最影响单位经济（CAC、LTV、毛利）？
3. 活动是否可标准化并规模复制？

### 2.8 重要伙伴（KP）

**填写问题：**
1. 哪些能力外部合作比内部自建更优？
2. 是否存在“单点依赖”伙伴风险？
3. 伙伴关系是否具有战略排他性？

### 2.9 成本结构（C$）

**填写问题：**
1. 哪些成本是刚性成本？
2. 成本是否随规模下降（规模经济）？
3. 哪些投入短期看是成本、长期看是护城河？

## 4. VRIO 评估框架（用于资源/能力“质检”）

### 4.1 四个维度定义与评估标准

| 维度 | 定义 | 评估标准（可操作） |
|---|---|---|
| Value（价值） | 能否创造客户价值或提升效率 | 能否提高收入、降低成本、降低风险 |
| Rarity（稀有性） | 是否被少数竞争者掌握 | 行业内拥有者数量、可得性 |
| Inimitability（难模仿） | 竞争者复制难度与成本 | 复制时间、投入、专利、因果模糊性 |
| Organization（组织） | 企业是否具备利用该能力的体系 | 架构、流程、激励、人才匹配度 |

### 4.2 竞争含义判定表

| V | R | I | O | 竞争含义 |
|---|---|---|---|---|
| 否 | - | - | - | 竞争劣势 |
| 是 | 否 | - | - | 竞争对等 |
| 是 | 是 | 否 | - | 暂时竞争优势 |
| 是 | 是 | 是 | 否 | 未被充分利用优势 |
| 是 | 是 | 是 | 是 | 可持续竞争优势 |

### 4.3 VRIO 输出模板

```markdown
## VRIO评估
| 资源/能力 | V | R | I | O | 竞争含义 | 证据 |
|---|---|---|---|---|---|---|
| [能力A] | 是 | 是 | 是 | 是 | 可持续竞争优势 | [证据] |
| [能力B] | 是 | 否 | - | - | 竞争对等 | [证据] |
```

## 5. BMC + VRIO 到 SWOT(S/W) 的转化流程

### 5.1 步骤

1. 从 BMC 的 KR/KA/KP/CH 中提取候选能力清单。
2. 用 VRIO 对候选能力逐项判定竞争含义。
3. 将“可持续优势/暂时优势”映射到 SWOT 的 Strengths。
4. 将“竞争劣势/关键能力缺失”映射到 SWOT 的 Weaknesses。

### 5.2 转化模板

```markdown
## BMC+VRIO → SWOT(S/W)
| 候选能力 | VRIO结论 | SWOT映射 | 说明 |
|---|---|---|---|
| [算法能力] | 可持续竞争优势 | S1 | [说明] |
| [线下渠道能力] | 能力缺失 | W1 | [说明] |
```

## 6. 完整案例（示例）：小米生态链能力评估

### 6.1 BMC视角摘录

- **客户细分**：大众消费电子用户 + 智能家居用户。
- **价值主张**：高性价比 + 智能互联体验。
- **关键资源**：品牌、MIUI、IoT平台、生态链伙伴。
- **关键活动**：产品定义、生态投资、供应链协同、社区运营。
- **关键伙伴**：生态链公司、芯片/制造伙伴、电商渠道。

### 6.2 VRIO评估

| 资源/能力 | V | R | I | O | 结论 |
|---|---|---|---|---|---|
| 生态链协同能力 | 是 | 是 | 是 | 是 | 可持续竞争优势 |
| 品牌与粉丝社区 | 是 | 是 | 中高 | 是 | 暂时至可持续优势 |
| 供应链效率 | 是 | 否 | 否 | 是 | 竞争对等（但重要） |

### 6.3 SWOT映射

- **Strengths**：生态链协同、品牌社区心智。
- **Weaknesses**：部分高端技术自研深度不足（示例性假设）。

## 7. 最佳实践与常见误区

### 最佳实践

1. BMC先广覆盖，VRIO再深度筛选。
2. 每条 S/W 必须有证据，避免“感受型结论”。
3. 关注“组织维度(O)”——很多优势死在执行层。

### 常见误区

1. 把“行业标配能力”误判为优势。
2. 把“短期流量红利”误判为可持续护城河。
3. 只盘点资源，不验证组织是否能发挥资源价值。
