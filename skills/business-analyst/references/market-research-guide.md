# 市场调研指南

## 1. 调研流程

```
确定调研目标 → 设计调研方案 → 收集数据 → 分析数据 → 输出报告
```

## 2. 数据来源

### 2.1 一手数据
- 用户访谈
- 问卷调查
- 焦点小组
- 用户行为数据

### 2.2 二手数据

| 来源类型 | 具体来源 | 适用场景 |
|----------|----------|----------|
| 行业报告 | 艾瑞、易观、Gartner、IDC、麦肯锡 | 行业趋势、市场规模 |
| 政府数据 | 统计局、工信部、商务部 | 宏观经济、行业统计 |
| 企业财报 | 上市公司年报、招股书 | 竞品分析、行业数据 |
| 新闻媒体 | 36氪、虎嗅、TechCrunch | 行业动态、融资信息 |
| 学术研究 | 知网、Google Scholar | 深度研究、理论支撑 |

## 3. 市场规模计算方法

### 3.1 自上而下法 (Top-Down)
```
TAM = 行业总规模
SAM = TAM × 目标细分市场占比
SOM = SAM × 预期市场份额
```

### 3.2 自下而上法 (Bottom-Up)
```
SOM = 目标客户数 × 客单价 × 购买频次
SAM = SOM ÷ 预期市场份额
TAM = SAM ÷ 细分市场占比
```

```markdown

- 数据来源：[来源]

- 细分市场：[细分定义]

|------|----------|--------|
| 2024 | [金额] | - |
| 2025 | [金额] | [%] |
| 2026 | [金额] | [%] |
```

## 6. 行业趋势分析框架

| 趋势维度 | 分析要点 |
|----------|----------|
| 技术趋势 | 新技术应用、技术成熟度、技术壁垒 |
| 市场趋势 | 市场增长、细分变化、区域差异 |
| 用户趋势 | 需求变化、行为变化、偏好变化 |
| 政策趋势 | 监管政策、扶持政策、行业标准 |
| 竞争趋势 | 竞争格局、并购整合、新进入者 |
