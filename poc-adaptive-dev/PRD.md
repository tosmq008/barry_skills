# Product Requirements Document (PRD)

## 1. 项目简介
这是一个简单的 Python 命令行小工具（CLI）。允许用户通过命令行传递姓名，并将其友好地打印出 "Hello, [Name]" 样式的欢迎语。

## 2. 核心功能
* **接收参数**：能够解析命令行后的 `--name` 或非具名参数字符串。
* **打印输出**：向控制台标准输出 (stdout) 打印。
* **默认行为**：若不传入参数则默认使用 "World" 并打印 "Hello, World"。

## 3. 非功能性需求
* 需要附带完整的自动化单元测试。
* 具有基本的可运行启动配置/文档支撑。
